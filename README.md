# Motywator Pro 🚀

Aplikacja do śledzenia nawyków i budowania dyscypliny. Pozwala na codzienne zaznaczanie sukcesów lub porażek w wybranych obszarach życia.

## Funkcje

- **Śledzenie nawyków**: Bieganie, Oddechy, Papierixy, Trwa, Koń.
- **System punktowy**: Każdy dzień sukcesu to 1 punkt. Licznik resetuje się po zaznaczeniu relapsu (czerwony dzień).
- **Całkowity wynik**: Suma punktów ze wszystkich śledzonych nawyków.
- **Interaktywny kalendarz**: Podgląd ostatnich 14 dni z możliwością szybkiego przełączania statusu (Brak -> Sukces -> Relaps).
- **Zapis danych**: Twoje postępy są zapisywane lokalnie w przeglądarce (`localStorage`).

## Technologie

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (ikony)
- **date-fns** (operacje na datach)

## Design

Aplikacja posiada nowoczesny, mroczny interfejs ("Dark Mode") z neonowymi akcentami (zielony dla sukcesu, czerwony dla relapsu, niebieski dla statystyk), zaprojektowany tak, aby motywować do działania i być czytelnym na każdym urządzeniu.

## Rozwój

Aplikacja jest przygotowana pod przyszłą integrację z serwerem w celu synchronizacji danych między urządzeniami.
